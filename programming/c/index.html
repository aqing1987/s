C Languages Notes


C (/ˈsiː/, as in the letter c) was originally developed by Dennis Ritchie between 1969 and 1973 at Bell Labs, and used to re-implement the Unix operating system.

Despite its low-level capabilities, the language was designed to encourage cross-platform programming.

It's still the main language for embedded programming now.
